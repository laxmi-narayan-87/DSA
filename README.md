# Data Structures and Algorithms (DSA)

LeetCode solutions and reference material for data structures and algorithms.

## Structure

```
DSA/
├── [problems/](./problems/)         # All LeetCode content
│   ├── [Python3/](./problems/Python3/)      #   Solutions by language & difficulty
│   ├── [C++/](./problems/C++/)
│   ├── [MySQL/](./problems/MySQL/)
│   ├── [2026/](./problems/2026/)         #   Daily challenge notes
│   └── [leetcode/](./problems/leetcode/)     #   Individual problem writeups
├── [algorithms/](./algorithms/)       # Algorithm explainers (sorting, etc.)
└── [data-structures/](./data-structures/)  # DS explainers (trees, arrays, etc.)
```

Each problem is organized as `problems/<Language>/<Difficulty>/<problem-slug>/`. Multiple solution files in a problem folder represent different approaches or attempts.
